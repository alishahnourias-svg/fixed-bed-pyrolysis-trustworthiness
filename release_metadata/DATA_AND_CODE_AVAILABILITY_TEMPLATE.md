# Manuscript availability statements

## Data availability

The code, data dictionary, feedstock-family mapping, group and feature-protocol definitions, fold-level metrics, aggregate uncertainty and applicability-domain diagnostics, eligibility summaries, final screening windows, manuscript tables, figure-source data, and figures are openly available in Zenodo at [PUBLIC DOI]. Processed row-level analytical datasets, record-level predictions, conformal intervals, applicability-domain scores, and candidate-level scenario tables are archived under restricted access at [RESTRICTED DOI] and may be made available for peer review and reproducibility verification subject to the stated access conditions. The original source workbooks were obtained from the supplementary information of Abaei et al. (DOI: 10.1021/acs.iecr.5c03390) and are not redistributed.

## Code availability

The complete Python workflow for cleaning and harmonization, feature construction, grouped validation, conformal calibration, applicability-domain mapping, eligibility auditing, scenario generation, reliability-aware screening, and manuscript-output generation is openly archived at [PUBLIC DOI]. The development repository is available at [GITHUB URL].
